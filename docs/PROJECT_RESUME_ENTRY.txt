Title: Enterprise Healthcare Revenue Cycle Management (RCM) Data Platform

Description: Engineered a production-grade data platform on Azure using a Medallion Architecture (Bronze, Silver, Gold) to ingest and process complex clinical and financial healthcare datasets. Leveraged Terraform for Infrastructure as Code, Azure Data Factory for orchestration, and Databricks for Spark-based transformations. Implemented reliable Delta Lake processing patterns and automated CI/CD via GitHub Actions.
