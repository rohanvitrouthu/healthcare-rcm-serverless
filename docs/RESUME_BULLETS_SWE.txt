# Software Engineering Focused Resume Bullet Points - Healthcare RCM Project

*   **Architected Cloud-Native Infrastructure:** Designed a scalable Azure environment using **Terraform**, achieving **100% Infrastructure-as-Code (IaC) coverage** for deploying Databricks, SQL databases, storage, and Key Vault resources.
*   **Engineered Containerized Microservices:** Developed **5+ decoupled Python microservices** (including API extractors and data simulators) to generate and process **30,000+ complex clinical records** (Patients, Encounters, Claims) and simulate high-volume enterprise workloads.
*   **Built Automated CI/CD Pipelines:** Implemented **GitHub Actions** workflows that reduced deployment time by automating Docker builds, unit testing, and infrastructure synchronization, ensuring **zero-downtime** updates for the data platform.
*   **Developed Resilient Backend Logic:** Wrote robust, object-oriented Python code for asynchronous data ingestion and validation, integrating with **Airflow** to manage complex dependency chains and ensuring **99.9% data integrity** across Bronze/Silver/Gold lakehouse layers.
*   **Implemented Observability & Cost Optimization:** Deployed a full-stack **Prometheus & Grafana** monitoring suite for real-time cluster metrics and authored a custom auto-shutdown script that **eliminated idle cloud costs** by detecting inactivity windows of 10 minutes.
